Getting Started

The Mindset

Learning web development in the present can be tricky. That’s frustrating, because it really isn’t all that difficult. It can be confusing at times, sure, but the content really isn’t the problem; rather, we proffer that the more fundamental issue is the teaching tools currently available.

People wishing to learn web development are faced with conflicting opinions about where to start, and the slurry of some-better-than-others teaching tools that are readily available online can, at times, be overwhelming.

Our goal is to take the complication out of learning web development. We aim to demystify the process as a whole as we direct users through a linear path of curated resources and projects. Our resources come from various sites across the Internet-- the trick is that we chose the best ones. This saves you, the user, from having to sift through the confusing, at times, inaccurate teaching tools that are out there in order to learn quickly and effectively.

The Setup

First things first, you will need a text editor in order to write your programs. A text editor is the simplest environment where you can produce code.

We recommend Sublime Text. This is a powerful and commonly used text editor that provides a straightforward way to get started.

Sublime Text is our editor of choice.

You may have heard of IDEs (Integrated Development Environemnts), which are another resource commonly used by software engineers to produce computer programs. We suggest that you start out with a text editor. There are many reasons for this.

  • A text editor is lightweight, and requires much less memory than most IDEs.
  • Using a text editor will force you to write out most of your code, instead of having it be completed automatically.
  • A text editor provides a low-distraction environment.

How Our Projects Work

Once you’ve gotten past the initial resources described above, we’ll have you go through a project to test your wings a bit by having you implement a practical piece of code using the knowledge you’ve gained.

Our step-by-step approach to learning new concepts

This section will basically give you a list of questions that you should be able to answer after having gone through the resources. They’re usually multiple choice, and it’s no big deal if you get them wrong. The main point of the questions is to point you in the right direction for where to go with your code; between questions, you should be bouncing back to Sublime Text or whatever tool you’re using to write your code and actually begin implementing your project there.

Also, don’t worry if a question is stumping you! After two wrong answers, we’ll give you the option to actually check out the solution code, so you should never feel like you don’t know how to move forward. With that said, we’d suggest that you start off with our Design project.

Happy coding!
